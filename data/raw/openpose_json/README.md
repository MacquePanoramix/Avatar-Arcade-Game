# OpenPose JSON Captures

Store one take per folder under each gesture/person/session path.

- One take = one folder containing OpenPose JSON frame files.
- Suggested take naming: `take_001`, `take_002`, etc.
- Keep one person in frame for each take.
- Start and end every take in a neutral pose.
- If a matching RGB video exists, store it under `data/raw/rgb_video`.
